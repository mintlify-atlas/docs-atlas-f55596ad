# Welcome to Robotics School

Robotics School is a comprehensive Learning Management System (LMS) built with Laravel 12, designed specifically for robotics education institutions. This platform empowers teachers to create and manage courses while providing students with an organized learning environment.

## What is Robotics School?

Robotics School was created to help small robotics schools manage their educational programs more effectively. The system provides a centralized platform where teachers can create courses, manage didactic materials, and organize students into learning groups based on their skill levels.

## Core Components

The platform is structured around three main entities:

### Users (Usuarios)

The system supports three distinct user roles:

- **Students**: Access assigned courses and learning materials
- **Teachers**: Create and manage courses, upload materials, and organize content
- **Administrative**: Oversee platform operations and user management

Each user has a unique email address and can be assigned to specific groups for organized learning paths.

### Courses (Cursos)

Courses are the heart of the platform, featuring:

- **Title**: Clear, descriptive course names
- **Cover Image**: Visual representation of the course
- **Content**: Comprehensive course material and curriculum
- **Didactic Material**: Supporting educational resources
- **Robotics Kit**: Specification of required physical robotics kits

Courses can be assigned to multiple groups, allowing flexible curriculum design.

### Groups (Grupos)

Groups enable skill-level organization:

- **Beginner**: Foundation courses for new students
- **Intermediate**: Progressive learning for developing skills
- **Advanced**: Complex projects and advanced robotics concepts

The many-to-many relationship between courses and groups provides maximum flexibility in curriculum planning.

## Architecture Highlights

### Database Schema

The platform uses a relational database structure with the following tables:

- `usuarios`: User accounts with role-based access
- `grupos`: Learning groups for organizing students
- `cursos`: Course catalog with all educational content
- `course_groups`: Junction table for many-to-many course-group relationships
- `cache`, `jobs`: Laravel system tables for caching and queue management

### Development Features

The platform includes several development conveniences:

- **Automated Setup**: One-command installation script
- **Concurrent Development**: Run server, queue, logs, and Vite simultaneously
- **Testing Suite**: Integrated PHPUnit testing framework
- **Code Quality**: Laravel Pint for code style enforcement
- **Real-time Logs**: Laravel Pail for log monitoring

<Warning>
