## Key Features

Robotics School provides a complete platform for managing robotics education programs.

<CardGroup cols={2}>
  <Card title="Role-Based Access" icon="user-shield" href="/features/role-based-access">
    Three user roles: Students, Teachers, and Administrative staff with appropriate permissions
  </Card>
  <Card title="Course Management" icon="book" href="/features/course-management">
    Create and manage courses with content, cover images, didactic materials, and robotics kits
  </Card>
  <Card title="Group Organization" icon="users" href="/features/group-system">
    Organize students into skill-level groups: beginner, intermediate, and advanced
  </Card>
  <Card title="Laravel Framework" icon="code" href="/development/architecture">
    Built on Laravel 12 with modern PHP architecture and best practices
  </Card>
</CardGroup>

## Quick Start

Get your Robotics School instance up and running in minutes.

<Steps>
  <Step title="Install Dependencies">
    Clone the repository and install PHP and JavaScript dependencies using Composer and npm.
  </Step>
  <Step title="Configure Environment">
    Set up your database connection and application settings in the `.env` file.
  </Step>
  <Step title="Run Migrations">
    Create the database schema and seed with default users for testing.
  </Step>
</Steps>
