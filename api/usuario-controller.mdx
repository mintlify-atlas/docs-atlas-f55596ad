## Overview

The `UsuarioController` handles all CRUD operations for user resources in the Robotics School application. This controller follows Laravel's resource controller pattern.

**Namespace:** `App\Http\Controllers`

**Extends:** `Controller`

<Warning>
  The controller methods are currently empty placeholders. You will need to implement the logic for each method and register the resource routes in `routes/web.php` using:
  
  ```php
  Route::resource('usuarios', UsuarioController::class);
  ```
</Warning>

---

## Methods

### index()

Display a listing of user resources.

```php
public function index()
```

**Purpose:** Retrieve and display all users in the system.

**Parameters:** None

**Returns:** View or JSON response with list of users

**HTTP Method:** GET

**Route:** `/usuarios`

---

### create()

Show the form for creating a new user resource.

```php
public function create()
```

**Purpose:** Display the form for creating a new user.

**Parameters:** None

**Returns:** View with user creation form

**HTTP Method:** GET

**Route:** `/usuarios/create`

---

### store()

Store a newly created user resource in storage.

```php
public function store(Request $request)
```

**Purpose:** Validate and save a new user to the database.

<ParamField path="request" type="Request" required>
  The HTTP request object containing user data from the creation form.
</ParamField>

**Returns:** Redirect to user list or show page, or JSON response with created user

**HTTP Method:** POST

**Route:** `/usuarios`

### show()


**Purpose:** Retrieve and display a specific user by ID.

<ParamField path="id" type="string" required>

</ParamField>

**Returns:** View or JSON response with user details

**HTTP Method:** GET

**Route:** `/usuarios/{id}`
