## Overview

The `CursoController` handles all CRUD operations for course resources in the Robotics School application. This controller follows Laravel's resource controller pattern.

**Namespace:** `App\Http\Controllers`

**Extends:** `Controller`

<Warning>
  The controller methods are currently empty placeholders. You will need to implement the logic for each method and register the resource routes in `routes/web.php` using:
  
  ```php
  Route::resource('cursos', CursoController::class);
  ```
</Warning>

---

## Methods

### index()

Display a listing of course resources.

```php
public function index()
```

**Purpose:** Retrieve and display all courses in the system.

**Parameters:** None

**Returns:** View or JSON response with list of courses

**HTTP Method:** GET

**Route:** `/cursos`

---

### create()

Show the form for creating a new course resource.

```php
public function create()
```

**Purpose:** Display the form for creating a new course.

**Parameters:** None

**Returns:** View with course creation form

**HTTP Method:** GET

**Route:** `/cursos/create`

---

### store()

Store a newly created course resource in storage.

```php
public function store(Request $request)
```

**Purpose:** Validate and save a new course to the database.

<ParamField path="request" type="Request" required>
  The HTTP request object containing course data from the creation form.
</ParamField>

**Returns:** Redirect to course list or show page, or JSON response with created course

**HTTP Method:** POST

**Route:** `/cursos`

---

### show()

Display the specified course resource.

```php
public function show(string $id)
```

**Purpose:** Retrieve and display a specific course by ID.

<ParamField path="id" type="string" required>
  The unique identifier of the course to display.
</ParamField>

**Returns:** View or JSON response with course details

**HTTP Method:** GET

**Route:** `/cursos/{id}`

---

### edit()

```php

```

**Purpose:** Display the form for editing an existing course.

<ParamField path="id" type="string" required>
  The unique identifier of the course to edit.

### update()

Update the specified course resource in storage.

```php
public function update(Request $request, string $id)
```

**Purpose:** Validate and update an existing course in the database.

<ParamField path="request" type="Request" required>
  The HTTP request object containing updated course data from the edit form.
</ParamField>

<ParamField path="id" type="string" required>

</ParamField>

### destroy()

Remove the specified course resource from storage.

public function destroy(string $id)

<ParamField path="id" type="string" required>
